Classical Ciphers

Named after the muse of History, ClioCrypt is a web app
about classical ciphers. In cryptography, a classical cipher
is one that was used historically but now has fallen into disuse.

Caesar Cipher

A monoalphabetic substitution cipher in which each letter in the plaintext is replaced by a letter some fixed number of positions down the alphabet, wrapping around it. For example, with a key of 3, A would be replaced by D, and so on; with a key of 27, Z would be replaced by A. Thus, Caesar's algorithm can be generalised as:

ciphertext[i] = (plaintext[i] + key) mod 26

The method is named after Julius Caesar, who was the first recorded user of the scheme. According to Roman Empire historian Suetonius, Caesar used it with a shift (key) of 3 to protect messages of military significance.


Vigenère Cipher

A polyalphabetic substitution cipher in which each letter in the plaintext is shifted a la Caesar by a different value, each corresponding to a letter of a keyword. For example, with a key of "ay", the first letter of the plaintext would be shifted by 0 (a), the second by 25 (y), the third by 0 again, and so on.

The method is named after Blaise de Vigenère, although that is a misattribution. Blaise de Vigenère did publish a type of polyalphabetic cipher in 1586, but the "Vigenère cipher" is the one first described by Giovan Battista Bellaso in his 1553 book La cifra del Sig. Giovan Battista Bellaso.


Scytale Cipher

A transposition cipher in which a band is wrapped around a scytale – a cylinder of some fixed size –, a message is written, and when the band is unrolled a ciphertext appears.

Without a scytale, the same is achieved by placing the plaintext row-wise on a grid some fixed number of columns, then rewriting it column-wise. For example, with a key of 4, "helloworld" becomes "holewdloxlrx" (with X as filler):



The method was used in Ancient Greece and Sparta.
